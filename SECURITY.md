# Security Policy

Folines is in early development.

## Reporting a Vulnerability

Please report security concerns by email:

security@folines.com

Include the affected URL or build, reproduction steps, expected behavior, and observed behavior.

## Current Product Boundary

This repository contains the public Folines marketing site. It does not contain the commercial Windows app source.

The Folines desktop app is designed to:

- run as a standalone Windows app;
- avoid any dependency on Lively Wallpaper;
- store wallpaper settings locally;
- avoid account requirements for core wallpaper usage.
