# Folines Site

Static launch site for `folines.com`.

Folines is an independent standalone Windows product for focus-oriented live typography wallpapers.

## Current download

- Public artifact: `downloads/Folines-0.3.11-win-x64-early-access.zip`
- Build type: early-access Windows ZIP
- Signing state: unsigned alpha; Windows may warn until the signed installer pipeline is ready
- Install path after running the included installer script: `%LOCALAPPDATA%\Programs\Folines`

## DNS target

Use the records in `DNS_NAMECHEAP.md` only after the dedicated Folines hosting target is ready.
